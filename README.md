# More-Meals
A recipe finder using HTML, CSS & Vanilla JS with the ![TheMealDB](https://www.themealdb.com/).

Displays meal name, ingredients and instructions.

# Features:
- Search by ingredient
- Search by meal name
- Random Meal Ideas
- View ingredients and instructions for each meal
